StatiGram - Install
Steps:

1. Upload the statigram folder to your web site.

2. Edit the "config.php" (in the admin folder), and "createtables.php" (in the install folder).

3. Upload "config.php" and "createtables.php" files.

4. Go to "/install/index.html" (which is here).

"/install/" Steps:

1. Create tables. CLICK HERE (ONLY OPEN THIS LINK ONCE!)

2. Delete the "/install/" directory (This is important for security reasons!)

3. Finish!

Your Statigram should now be working!

To get into the admin area go to www.yourdomain.com/statigramdirectory/

The password is the password you set in the "createtables.php" file (default: aniket).

To get visitor stats for a page, include the db_functions.php file and the code on any page on your website, (Just copy and paste) using:

<?php require('statigram/db_functions.php');
$db = new db_functions();
$referer=$_SERVER['HTTP_REFERER'];
$db->storeEverything($referer);
?>

and copying the db_functions.php file into the same

directory as any pages including it.
If you want to track the html pages you will have to add following to your .htaccess file AddType application/x-httpd-php .html .htm if you are using php5 then include this AddType application/x-httpd-php5 .html .htm and if either of this don't work then you will have to rename your html file to php.
